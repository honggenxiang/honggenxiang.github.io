Elasticsearch 7.8.0 调试环境搭建

1.环境准备

  • Elasticsearch 源码版本 7.8.0
  • jdk版本 14.0.1
  • Gradle 6.5.1
  • IDEA 2020.1
  • OS 为 macOS Catalina Version 10.15.5

2.源码下载

3.导入IDEA

参考文档:https://github.com/elastic/elasticsearch/blob/v7.8.0/CONTRIBUTING.md

  1. 选择 文件->打开
  2. 在文件选择框中选择Elasticsearch的根目录，及build.gradle所在目录
  3. 在接下的弹框中选择以项目打开

4.运行

启动类为模块 server的类 org.elasticsearch.bootstrap.Elasticsearch

异常1

elasticsearch启动需要elasticsearch.yml以及jvm等配置。为了方便，可官网下载 7.8.0 版本的 elasticsearch 解压，将es.path.confg配置到根目录中的 config 文件夹
